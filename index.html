One Tool to find them all

BioFragmentor is a python based tool for doing in silico fragmentation of biomolecules, such as RNA, DNA or proteins. Due to the usage of external XML datafiles, the number of supported sequence types can easily be extended to other polymers abiding to a few simple rules, such as beeing linear or having the same ligation handle (like a amide group) between them.

Requirements

  • Python 3

Usage

Display Help: $ python biofragmentor.py --help General usage: $ python biofragmentor.py [--type {protein,rna,dna}] [--mode {positive,negative}] [--maxcharges MAXCHARGES] SEQUENCE Create Fragments for a DNA tetramer with the sequence 5'-TACG-3' without leading or ending phosphate groups: $ python biofragmentor.py --type=dna TACG Create Fragments for a RNA tetramer with the sequence 5'-UACG-3' without leading or ending phosphate groups $ python biofragmentor.py --type=rna UACG DNA trimer with a leading phosphate $ python biofragmentor.py --type=dna pTAC Protein, with a maximum charges of +1 $ python biofragmentor.py --type=protein MGAP --maxcharge=1 --mode=positive
